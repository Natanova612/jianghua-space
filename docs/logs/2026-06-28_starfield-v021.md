## 完成内容

### 1. Header HUD 仪器化（StarLayout.astro）

| 要求 | 实现 |
|------|------|
| 1px 边框 rgba(74,158,255,0.3)，圆角 4px | `rounded border border-starfield-ai-blue/30` |
| faint grid 背景，0.5px 线宽，rgba(74,158,255,0.08)，20px 间距 | `linear-gradient` background-image + `background-size: 20px 20px` |
| 数字等宽 'Courier New' | inline `font-family: 'Courier New', monospace` |
| 标签 letter-spacing: 0.1em | Tailwind `tracking-[0.1em]` |
| 标签和数字用 │ 分隔，颜色 rgba(0,212,170,0.5) | 静态 `│` 字符 + `text-starfield-ai-cyan/50` |
| 底部 [SYNC: YYYY.MM.DD HH:MM] | 静态 `[SYNC: 2026.06.25 16:42]` |
| 在线光点外发光 | `box-shadow: 0 0 8px rgba(0,255,128,0.6)` |
| max-width 600px 居中 | `max-w-[600px] mx-auto` |

布局改为窄面板：在线状态 + 实体名 + 版本 → 分隔线 → 三项垂直排列的仪器读数 → 分隔线 → SYNC 时间戳 + 实时时钟。

顶部 `pt-48 md:pt-52` 为仪器面板留出空间。

### 2. 实验说明书横向 Tabs（starfield.astro）

- 移除原来的三栏 Accordion 组件。
- 新增横向标签栏：
  - 🧬 这是什么？
  - ⚡ 为什么值得关注？
  - 🧭 怎么阅读？
- 样式：
  - 默认：`text-starfield-human-silver/50` + 1px 下划线 `rgba(74,158,255,0.2)`
  - 激活：`text-starfield-human-silver` + 加粗 + 2px `#4a9eff` 下划线
  - 使用 `box-shadow: inset` 实现下划线，避免布局跳动
- 点击直接切换面板，无手风琴动画。
- 默认激活"这是什么？"。
- 每个面板内容压缩到 ≤3 句话 + 1 个小视觉元素：
  - Tab 1：一句话 + 时间线刻度
  - Tab 2：两句话 + 4 个疑问标签
  - Tab 3：两句话 + 路径 A/B 小卡片
- 面板容器使用与 Header 一致的 `border-starfield-ai-blue/30` 仪器边框。

### 3. 脚本更新

- 删除 Accordion 的展开/折叠逻辑。
- 新增 `activateTab()`：切换 tab 下划线样式 + 显示/隐藏对应 panel。

### 4. 构建验证

## 关键决策

1. **Header 由全宽导航改为居中仪器面板**：从"网站 banner"转向"发光数据面板"，参考 Rabbit.tech 头部体感。
2. **Tab 下划线用 `box-shadow: inset` 而非 `border-bottom`**：避免激活态 2px 与非激活态 1px 导致的高度不一致和布局跳动。
3. **内容严格控制在 3 句话 + 1 视觉元素**：减少认知负担，强化仪器界面的克制感。
4. **保留日期字段但面板内精简时间信息**：不修改 JSON，只在展示层把 Manual 时间线改为更克制的刻度。
