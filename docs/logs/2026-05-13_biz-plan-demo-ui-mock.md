## 技术实现

- 纯 Tailwind CSS + HTML，无外部图片资源
- 使用 `aspect-video` 保持 16:9 比例
- 深色主题与站点整体风格统一（bg-slate-900 底色 + 玻璃质感卡片）
- 响应式：手机框在小屏幕下自适应缩放

## 关键决策

1. **不用截图，用 CSS 画**：产品尚未开发，截图不存在；CSS 手绘更灵活，后续产品迭代后只需替换代码即可
2. **具体信息全部 grounded**：三个界面中的文字、数字、标签全部来自随山居档案，无泛化表达
3. **信息密度控制**：Demo 界面以"一看就懂"为目标，每屏只展示一个核心功能点
