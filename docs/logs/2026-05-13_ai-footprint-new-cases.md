### 3. 页面布局改版：两栏结构

**问题**：原布局中AI画像（PROFILE）占用了过多纵向空间，案例列表被压到下方很远。

**解决方案**：桌面端改为两栏布局。

```
┌──────────────────────┬──────────────┐
│  应用案例（2/3主栏）   │  AI画像（1/3）│
│  筛选标签 + 案例卡片   │  ↓ sticky    │
│                      │  生活融合度    │
│                      │  画像摘要      │
│                      │  工具矩阵      │
└──────────────────────┴──────────────┘
```

**侧栏精简细节**：
- 进度条从 h-2 缩至 h-1.5，间距压缩
- 画像摘要4条改为精简版（去掉括号数据）
- 工具矩阵从4列表格改为简洁列表（工具名 + 状态点 + 频次标签）
- 侧栏使用 `md:sticky md:top-24`，滚动时固定

### 4. 案例标题样式优化

**问题**：`font-human`（Long Cang 书法体）在小字号下笔画较细，在卡片中不够显眼。

**解决方案**：
- 添加 `font-bold` 加粗
- 添加 `inline-block px-2 py-0.5 rounded bg-amber-100/40` —— 极淡琥珀色底色块
- 保留 `group-hover:text-aurora-green` 交互

### 5. 自检清单表格渲染升级

**问题**：c-007 中的 Markdown 表格语法在弹窗中显示为纯文本，没有表格结构。

**解决方案**：
- 将 Markdown 表格替换为 HTML `<table class="case-table">`
- 添加 `.case-table` CSS 样式：表头浅灰底色、单元格边框、行 hover 高亮
- 风险等级添加颜色类：`.risk-high`（红）、`.risk-mid`（橙）、`.risk-low`（琥珀）
