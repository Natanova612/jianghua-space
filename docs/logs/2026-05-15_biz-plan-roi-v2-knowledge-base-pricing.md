### 定价策略重构：从"渐进式三阶段"到"两步定价"

**旧方案**：免费试用 → 效果付费（按单抽成）→ 订阅制（¥299/599/1299）

**新方案**：知识库构建服务（必须前置） + SaaS订阅（知识库完成后）

#### 第一步：知识库构建服务（独立报价单元）

| 方案 | 价格 | 包含内容 |
|------|------|----------|
| 一次性交付 | ¥3,000-8,000 | LLM自动提炼OTA描述+聊天记录、团队人工校准、5大场景初始搭建 |
| 分期方案 | ¥1,500/月 × 3个月 | 密集构建期 + Starter版SaaS + 1对1客户成功经理陪跑 |

**定位升级**：
- 旧：知识库是SaaS的附属功能
- 新：**"这不是安装费，是民宿的数字化底座"**
- 逻辑：没有知识库，AI店长就是空壳；有了这个底座，老板才能做任何AI深化

#### 第二步：SaaS订阅（¥499起步）

| 档位 | 月费 | 功能 |
|------|------|------|
| Starter | ¥499/月 | 智能客服 + 知识库维护 + 经营看板 + 老客档案 |
| Professional | ¥999/月（推荐） | +动态定价 + 竞品监控 + 内容生成 + 私域运营 |
| Enterprise | ¥1,999/月 | +收益分成 + API开放 + 专属客户成功经理 |

**渐进路径**：免费试用（14天）→ 知识库构建（一次性¥3,000-8,000）→ 效果付费过渡（可选）→ 订阅制（¥499起）

### ROI测算v2：基于"优质精品民宿"重新建模

**前提调整**：
- 目标客户从"月均营收2.5万、利润几乎为零"改为"月均营收3-5万、经营状态良好、高学历兼职经营者"
- 老板娘时间价值从¥30-50/小时上调至¥50-80/小时（高学历兼职经营者的机会成本）

**新测算模型**：

| 价值维度 | 月价值 | 计算逻辑 |
|----------|--------|----------|
| 重复咨询回复 | ¥2,000-3,200 | 每天省100分钟×30天×¥50-80/小时 |
| 老客维护 | ¥1,200-2,400 | 每月多1-2组复购×¥800-1,200/组 |
| 内容营销 | ¥600-1,200 | 每周多发3-4条带来的增量 |
| 动态定价/竞品监控 | ¥400-800 | 少错过旺季涨价窗口 |
| 知识库沉淀 | 长期价值 | 结构化资产，离职不流失，新人快速上手 |
| **合计** | **¥4,200-7,600** | |

**成本拆解**：
- 知识库构建（一次性¥5,000，按36个月摊销）：~¥140/月
- SaaS订阅：Starter ¥499/月 或 Pro ¥999/月
- **总成本：¥640-1,140/月**

**净收益**：
- Starter方案：¥4,200-7,600 - ¥640 = **¥3,560-6,960/月**
- Pro方案：¥4,200-7,600 - ¥1,140 = **¥3,060-6,460/月**

**ROI倍数**：
- Starter：投入¥640，回报¥4,200+，ROI约 **6.5-12倍**
- Pro：投入¥1,140，回报¥4,200+，ROI约 **3.7-6.7倍**

## 关键决策

1. **知识库独立报价的合理性**：用户明确指出"没有这个单元，老板是无法做任何AI深化的事情"——这意味着知识库构建不是可选项，是必选项。必选项就有定价权，不应该被SaaS订阅的价格锚定。
2. **¥499起步而非¥299**：目标客户是"经营状态好、学历高、科技接受度高"的精品民宿，支付能力更强，对价格敏感度更低。¥499/月对于月均营收3-5万的民宿来说是可接受的（占营收1-2%）。
3. **老板娘时间价值上调至¥50-80/小时**：高学历兼职经营者（互联网/金融背景）的机会成本远高于普通服务业从业者。这直接推高了ROI测算中的"省时价值"。
4. **知识库构建按36个月摊销**：一次性投入¥3,000-8,000看起来高，但分摊到3年只有¥80-220/月。这比持续雇佣一个兼职客服便宜得多。
